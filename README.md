# 基于 ESP32S3 的原穿戴设备端

> ⚠️ 由于特殊原因，项目已经不在使用，此项目仅供学习参考

基于 ESP-IDF 开发的穿戴式监控设备，通过 MQTT 协议与设备端进行数据交换。

## 功能概述

- **姿态检测**：MPU6050 采集用户头部两个自由度方向数据
- **实时通信**：MQTT 协议发布姿态数据、订阅图像数据
- **图像显示**：ST7789 LCD 屏幕显示设备端回传的实时画面
- **网络连接**：支持 WiFi STA/AP 模式

## 硬件配置

| 模块 | 型号 | 说明 |
|------|------|------|
| 主控 | ESP32-S3 | 双核 240MHz，支持 PSRAM |
| 显示屏 | ST7789 | 240x240 SPI 接口 |
| 姿态传感器 | MPU6050 | 三轴加速度 + 三轴陀螺仪 |

## 设计特点

- **SPI-DMA 乒乓缓冲**：可通过条件宏启用/禁用
- **PSRAM 支持**：各类缓冲区可分配到 PSRAM
- **独立移植 LVGL**：不使用 ESP 官方 LVGL 组件
- **模块解耦**：各组件独立，便于移植和扩展
